### 一、存放类型的类
（1）创建一个模块，在其中定义一个类，该类用于存放测试文件中所有的类型，包括基础类型以及自己定义的类的类型
（2）将所有基础类型存放到一个列表中（目前采用的是，存放的是基础类型的形式而不是字符串形式，觉得后期需要改进）
（3）对测试文件进行分析，将所有不在列表中的类型存到列表中（主要是自定义的类）
（4）对每个文件中导入的模块中，定义的类
（5）如果运行中，遇到新的类型，也存放到列表中
